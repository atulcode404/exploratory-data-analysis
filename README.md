# Exploratory-Data-Analysis-Project

# Customer Shopping Behavior Analysis using Pandas

## Project Overview

This project performs Exploratory Data Analysis (EDA) and Data Cleaning on a Customer Shopping Behavior dataset using Python and Pandas.

The goal is to understand customer purchasing patterns, clean inconsistent data, handle missing values, and extract meaningful insights from the dataset.

---

## Dataset

The dataset contains customer shopping information such as:

* Customer details
* Item Purchased
* Product Category
* Purchase Amount (USD)
* Review Rating
* Previous Purchases
* Product Size
* Other shopping-related attributes
